- General
 - cmake

 - fftw  
- Graphs file output 
 - libpng
 - cairo (optional)  
- Documentation 
 - doxygen
 - graphViz
  


## Intallation


### Linux & Mac 
 
    mkdir build ; cd build ; cmake ..
    make
    sudo make install

Optional :

    make doc
